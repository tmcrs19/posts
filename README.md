## Getting Started
Vercel deployment (socket connection won't work here, run locally for that)
https://posts-2hmc0o7u8-tmcrs19s-projects.vercel.app/

How to run locally:

```bash
nvm use
npm i
mkcert localhost
npm run dev
```

Alternatively try the script
```bash
chmod +x setup.sh
./setup.sh
```

Test
```bash
npm run test
```
